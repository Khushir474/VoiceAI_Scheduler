# DOPS-16: On-demand calendar context fetch for a specified date

Add a tool to the conversation agent that fetches full calendar context for any date the user asks about during the call. Example: 'What do I have on Friday?' or 'What's on my calendar next Monday?' — Max fetches and summarizes that day's events on demand. Distinct from DOPS-15 (which covers the fixed present-day context loaded at call start). Tool accepts a date parameter (relative: 'tomorrow', 'next Friday'; or absolute: 'July 4th'). Fetches from both Google Calendar and Apple iCal adapters. Returns normalized CalendarEvent list for that date. Caches result within the session to avoid repeat API calls for the same date. Touches: agents/conversation_agent.py, adapters/calendar/google_calendar.py, adapters/calendar/apple_ical.py, agents/state.py. Acceptance: Tool available to Max during call; handles relative and absolute date references; fetches from both calendar sources; caches per-date within session; results logged to debug_logs with date queried and event count; graceful handling if no events found.
